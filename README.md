# 퍼블리싱 코드 자동 검수 도구

VCP(Vehicle Content Studio) 퍼블리싱 코드를 자동으로 검수하고, 수정이 필요한 부분을 찾아 보고서를 생성합니다.

## 사용 방법

### 1. 준비
```bash
pip install -r requirements.txt
```

### 2. 실행
```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

### 3. 검수
1. ZIP 파일(퍼블리싱 코드)을 업로드합니다
2. 검수 기준을 선택합니다
3. "검수 시작" 버튼 클릭
4. 결과 리포트를 확인합니다

---

## 검수 기준

| 번호 | 기준 | 설명 |
|------|------|------|
| 1 | 타이포그래피 | 폰트 크기/굵기/줄간격이 너무 많이 직접 입력된 경우 |
| 2 | 색상 | 색상 코드(#111111 등)가 너무 많이 반복된 경우 |
| 3 | !important | CSS `!important` 과다 사용 |
| 4 | 스크롤바 | 브라우저 호환성 없는 스크롤바 스타일링 |
